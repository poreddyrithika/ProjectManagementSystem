# Project Management System

## Overview

The Project Management System is a desktop application developed using Java Swing as part of the Object-Oriented Programming (OOP) course. The application provides a simple and interactive graphical user interface for managing project records, tracking project status, and storing data permanently using file handling.

## Features

* Add new project records
* Store project name, manager name, and project status
* Track project status (Ongoing, Completed, Pending)
* Delete project records
* Input validation for user entries
* Display project data using JTable
* Save project records to a text file
* Automatically load saved records when the application starts

## Technologies Used
